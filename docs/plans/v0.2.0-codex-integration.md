# TokenTrail V0.2.0 — Codex 接入计划

## 目标

让用户在接入指南中直接找到经过验证的 Codex 接入方式，并统一应用版本为 `V0.2.0`。

## MVP 范围

1. 新增独立的 Codex 接入页签。
2. 提供零配置本地扫描、独立预检/导入、可选 Skill 三种接入方式。
3. 所有命令与当前项目代码、当前 Codex 本地会话格式保持一致。
4. 同步更新页面、健康接口和根包版本。
5. 更新接入文档，并通过类型检查、生产构建和真实页面验证。

## 验收标准

- Codex 页签可见，命令可复制，窄屏下可横向滚动访问。
- `~/.codex/sessions/**/*.jsonl` 的真实 `token_count.last_token_usage` 能被现有同步器识别。
- 独立 Codex 扫描脚本的 `--dry-run` 能正常输出统计。
- 页面与 `/api/health` 均显示 `0.2.0`。
- 生产构建通过，接入弹窗在浏览器中正常显示。
